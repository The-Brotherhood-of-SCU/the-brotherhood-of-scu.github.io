# About

This is the blog of `The Brotherhood of SCU`

# Clone
This repo contains another git repo as a submodule.
To clone this repo, you need to add `--recursive` option.

```bash
git clone --recursive https://github.com/The-Brotherhood-of-SCU/the-brotherhood-of-scu.github.io.git
```
