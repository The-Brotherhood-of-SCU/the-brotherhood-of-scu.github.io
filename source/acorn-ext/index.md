# Acorn Extended

`Acorn-Ext` (Acorn Extended) 是`Acorn`的定制版本，是一种`hexo`主题，适用于organization。

相比[原版](https://acorn.imaging.xin/)：
- 修复了bug： 修复一些UI错位
- 增加了功能： Latex(Katex)支持等等
- 改动了界面： 移除价格、designer等UI
- 优化了逻辑： quotation组件无需硬编码序号等等
